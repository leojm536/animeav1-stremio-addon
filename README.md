# AnimeAV1 Stremio Addon

<p align="center"><img src="https://animeav1.com/img/logo-dark.svg" alt="AnimeAV1 logo" width="256"/></p>

Fork modificado del addon original de [Pigamer37](https://github.com/Pigamer37/animeflv-stremio-addon), adaptado **exclusivamente para AnimeAV1**. Esta versión simplificada carga el addon directamente desde Stremio sin configuración adicional.

> Basado en el trabajo original de Pigamer37 — solo modifiqué y recorté el addon para enfocarlo en AnimeAV1.

## Instalación en Stremio

Igual que el addon original, tienes dos opciones:

- Pega `https://tuservidor/manifest.json` en la barra de búsqueda de addons de Stremio
- O copia `stremio://tuservidor/manifest.json` en tu navegador

Reemplaza `tuservidor` con la URL donde tengas el addon corriendo.

## ¿Qué hace este addon?

- 📺 Catálogo de anime desde AnimeAV1
- 🔍 Búsqueda por nombre y filtro por género
- 📡 Fuentes de streaming desde AnimeAV1 directamente en Stremio

> ⚠️ Solo carga dos fuentes de reproducción directa dentro de Stremio: **YourUpload** y **MP4Upload**. Las demás fuentes abren el reproductor externo en el navegador.
